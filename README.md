# 🚀 Introdução ao Express.js

Um projeto simples desenvolvido para estudar os conceitos básicos do **Express.js**, incluindo:

- Criação de um servidor Node.js com Express  
- Uso do Router para gerenciamento de rotas  
- Organização do projeto utilizando o padrão **MVC (Model-View-Controller)**

---

## 📚 Sobre o projeto

Este projeto foi criado com o objetivo de **aprender e praticar os fundamentos do Express.js**, um framework minimalista para criação de aplicações web em Node.js.

Durante o desenvolvimento, foram abordados os seguintes pontos:

- Configuração inicial de um servidor Express;  
- Estruturação de rotas com o Express Router;  
- Separação da aplicação em **Routes** e **Controllers** para melhorar a organização e escalabilidade do código.

---

## ⚙️ Tecnologias utilizadas

- [Node.js](https://nodejs.org)  
- [Express.js](https://expressjs.com)

---

## 🚀 Como executar o projeto

### 1️⃣ Clonar o repositório
```bash
git clone https://github.com/MaghaiverGomesRamos/API-AprendizadoPratica.git
```

### 2️⃣ Instalar as dependências
```bash
npm install
```

### 3️⃣ Iniciar o servidor
```bash
npm run dev
```

O servidor estará rodando em:

👉 **http://localhost:8080**
